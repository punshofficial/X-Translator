# X Translator

Расширение для Chromium-браузеров, которое автоматически переводит посты в X на русский. Работает с основной лентой, ответами и цитируемыми постами, не затрагивая интерфейс сайта и имена пользователей.

Перевод заменяет исходный текст прямо в посте. Оригинал можно вернуть одной кнопкой. Ссылки, упоминания, хештеги, эмодзи и переносы строк сохраняются.

Исходный язык определяется автоматически — перевод работает не только с английским.

[Установить X Translator из Chrome Web Store](https://chromewebstore.google.com/detail/x-translator/jfillcfphbipmijdkcacnhjkoijcndll)

## Установка

Проще всего установить расширение из [Chrome Web Store](https://chromewebstore.google.com/detail/x-translator/jfillcfphbipmijdkcacnhjkoijcndll).

Для ручной установки:

1. Скачайте ZIP из [последнего релиза](https://github.com/punshofficial/X-Translator/releases/latest) и распакуйте его.
2. Откройте страницу управления расширениями и включите режим разработчика.
3. Нажмите «Загрузить распакованное расширение» и выберите распакованную папку.
4. Обновите открытую вкладку X.

## Важно

Расширение использует неофициальный веб-интерфейс Bing Translator без API-ключа. Если Bing изменит его или ограничит запросы, перевод может временно перестать работать.

В Bing отправляется только текст постов. Кэш, настройки и статистика хранятся локально в браузере.

Проверка проекта:

```bash
npm run check
npm run package
```
