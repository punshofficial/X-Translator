# Политика конфиденциальности X Translator

Последнее обновление: 19 июля 2026 года.

X Translator переводит текст постов, ответов и цитируемых публикаций непосредственно в интерфейсе X.

## Какие данные обрабатываются

Расширение считывает только текст публикаций, который необходим для перевода. Оно не собирает имена пользователей, данные учётной записи, пароли, файлы cookie, историю посещений, платёжные данные или содержимое личных сообщений.

Текст публикаций передаётся по защищённому HTTPS-соединению сервису Bing Translator, принадлежащему Microsoft, исключительно для получения перевода. Разработчик X Translator не получает и не хранит этот текст на собственных серверах и не использует его для рекламы, аналитики или иных целей.

Обработка данных сервисом Bing регулируется [заявлением Microsoft о конфиденциальности](https://privacy.microsoft.com/privacystatement).

## Локальные данные

Настройки расширения, кэш готовых переводов и локальные счётчики использования сохраняются только на устройстве пользователя через `chrome.storage.local`. Эти данные не передаются разработчику. Кэш можно очистить в окне расширения; все локальные данные удаляются при удалении расширения.

## Передача и продажа данных

X Translator не продаёт пользовательские данные и не передаёт их третьим лицам, кроме отправки текста публикаций в Bing Translator, необходимой для основной функции перевода. Данные не используются для персонализированной рекламы, определения платёжеспособности или любых целей, не связанных с работой расширения.

Использование данных X Translator соответствует требованиям ограниченного использования данных Chrome Web Store: данные используются и передаются только для предоставления заявленной функции перевода.

## Изменения политики

При изменении способов обработки данных эта политика будет обновлена до публикации соответствующей версии расширения.

Вопросы о конфиденциальности можно задать через [GitHub Issues](https://github.com/punshofficial/X-Translator/issues).
